# Open-source pattern review

Architecture and UX ideas were adapted; complete modules, keys, branding, environment files and financial logic were not copied.

- Mosleh92/exchange-platform: P2P filter/card and notification concepts.
- fairway-global/contra-clear: quote/expiry and activity feed.
- SpencerGeee/SpenTrade: timer, proof, timeline and conversation.
- ledgerleapllc/blockmatcher: review queues, batches and CSV export.
- openware/barong: KYC level/cap concept only; no Ruby code reused.

Live matching, custody, hot wallets, escrow, deposits, payouts and automatic release are out of scope.
